# Desktop Cleaner
### Install to:
Remember to install the program to:
```
/ProgramFiles/desktopcleaner_c
```
### Requirements:
- GCC Compiler
- Make (for makefile)
- Windows only (for now)

### Building and running:

<b>Build:</b>
```
make
```
<br><b>Compile:</b>
```
make compile
```
<br><b>Uninstall: </b>
```
make uninstall
```
<br><b>Running the program:</b>
```
./exec/desktopcleaner
```
